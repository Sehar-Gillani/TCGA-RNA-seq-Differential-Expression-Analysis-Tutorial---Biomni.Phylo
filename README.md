# 🧬 TCGA Bioinformatics Analysis using Python (Google Colab)

## 📌 Overview

This project demonstrates how to perform **bioinformatics analysis on TCGA (The Cancer Genome Atlas) data** using Python in Google Colab.

The goal of this project was not only to analyze biological data, but also to **learn bioinformatics practically** by combining:

* Hands-on coding in Python
* Real-world genomic datasets
* AI-assisted learning using **Biomni**

---

## 🚀 Learning Approach

Instead of following traditional static tutorials, I used **Biomni** as an interactive learning assistant to:

* Understand complex bioinformatics concepts step-by-step
* Break down genomic data analysis workflows
* Debug and improve Python code
* Explore new libraries and methods in real time

This made the learning process:

* More intuitive
* Faster
* Highly practical

---

## 🧪 What This Project Covers

The notebook includes:

* 📥 Accessing TCGA datasets
* 🧹 Data preprocessing and cleaning
* 📊 Exploratory data analysis (EDA)
* 🧬 Gene expression analysis
* 📈 Visualization of biological data
* 🧠 Interpretation of results
